# 📱 StudySync – Usability Testing Report

## Overview

This project presents a usability testing study conducted for **StudySync**, a student study planner application. The objective was to evaluate how effectively students could navigate and interact with the application's core features.

## Testing Objectives

* Evaluate application usability
* Identify navigation issues
* Assess task completion efficiency
* Gather user feedback for design improvements

## Test Tasks

1. Log in to the application
2. Add a new study task
3. Set a reminder for an assignment
4. View upcoming deadlines in the calendar
5. Check weekly study progress
6. Navigate to the profile screen

## Participants

* Engineering Student
* Final Year Student
* Computer Science Student
* Engineering Student
* College Student

## Key Observations

* Some users could not immediately locate the Add Task button.
* The calendar interface was easy to understand.
* Navigation icons appeared too small.
* Users found the progress screen useful.
* Some labels required better clarity.

## Metrics

| Metric                  | Result    |
| ----------------------- | --------- |
| Task Success Rate       | 90%       |
| Average Completion Time | 2 Minutes |
| User Satisfaction       | 4.5/5     |
| Navigation Errors       | 3         |

## Design Iterations

### Iteration 1

**Problem:** Add Task button lacked visibility.
**Solution:** Increased button size and improved visibility.

### Iteration 2

**Problem:** Navigation icons appeared too small.
**Solution:** Increased icon size and improved spacing.

### Iteration 3

**Problem:** Active navigation tab was unclear.
**Solution:** Added color highlighting for active navigation.

## Deliverables

* Usability Testing Report
* User Observations
* Metrics Analysis
* Design Iterations
* Before and After Comparisons

## Tools Used

* Figma
* Microsoft Word

## Conclusion

The usability testing process helped identify minor navigation and visibility issues within the StudySync application. Based on user feedback, several improvements were implemented to enhance usability and accessibility, resulting in a more intuitive and user-friendly experience.
